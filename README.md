# 🏆 TierMaker - 在线等级排序工具

自定义等级梯度，拖拽排序你的项目。

## ✨ 功能特点

- **自定义等级** - 自由创建等级（S/A/B/C/D、神/强/中/弱/拉等），支持自定义名称和颜色
- **拖拽排序** - 从项目池拖拽到对应等级，支持等级内排序和跨等级移动
- **批量添加** - 支持单个添加和批量添加排序项
- **快速预设** - 内置多种预设模板（SABCD、中文评级、Emoji、米其林等）
- **响应式设计** - 适配桌面端和移动端

## 🛠️ 技术栈

- **Vue 3** - 渐进式 JavaScript 框架
- **Vite** - 下一代前端构建工具

## 🚀 本地运行

```bash
npm install
npm run dev
```
